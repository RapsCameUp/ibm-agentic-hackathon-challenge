# Health AI Assistant

An intelligent health management system powered by IBM Watson that provides personalized dietary recommendations, calendar scheduling, and multi-channel notifications for patients.

## Overview

Health AI Assistant is an accessibility-first application that helps users manage their health through AI-powered dietary analysis and recommendations. The system features voice output for visually impaired users and seamless integration with popular platforms like Google Calendar, WhatsApp, and Slack.

LifeGuardian is a multi-agent wellness system powered by IBM watsonx Orchestrate that monitors your physical and mental health and takes automated, intelligent actions to improve your wellbeing. Unlike traditional wellness apps that only track data, LifeGuardian acts on behalf of the user through specialized AI agents.

# The Problem
Modern life brings unprecedented challenges:

- Chronic burnout from constant connectivity
- Poor sleep quality affecting mental and physical health
- Unhealthy lifestyle habits due to busy schedules
- Reactive healthcare instead of proactive wellness management
- Information overload without actionable insights

# The Solution
LifeGuardian combines proactive health detection with automated intervention. Using specialized agents covering nutrition, exercise, sleep, mental health, biometrics, and work-life balance, the system continuously interprets health signals and lifestyle patterns. IBM watsonx Orchestrate then executes personalized interventions: adjusting calendars, sending reminders, optimizing routines, booking appointments, and delivering real-time support.
Result: A comprehensive wellness manager that improves daily health outcomes, boosts productivity, and empowers individuals to maintain balance with minimal effort.
